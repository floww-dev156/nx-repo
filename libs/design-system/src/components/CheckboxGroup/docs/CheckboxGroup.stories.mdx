##Checkbox Group

This Doc will gives you a brief description of the Checkbox group
component.

Follow below code snippet to useCheckboxGroup and Checkbox components.

```tsx
import { Checkbox, CheckboxGroup } from '@nx-repo/design-system';

return (
  <CheckboxGroup
    label="Favorite pet Favorite pet"
    onChange={(value: string[]) => console.log(value)}
    {...props}
  >
    <Checkbox value="dogs">Dog</Checkbox>
    <Checkbox value="cats">Cats 1</Checkbox>
  </CheckboxGroup>
);
```

**CheckboxGroup Props**

```tsx
interface CheckboxGroupProps extends AriaCheckboxGroupProps {
  children: React.ReactNode;
  onChange: (value: string[]) => void;
  hint?: React.ReactNode;
  errorMessage?: string;
  isRequired?: boolean;
  containerClassName?: string;
  size: Small | Medium;
}
```

**Checkbox Props**

```tsx
interface CheckboxProps extends AriaCheckboxGroupItemProps {}
```
