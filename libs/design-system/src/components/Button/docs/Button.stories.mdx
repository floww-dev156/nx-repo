## Button

This Doc will gives you a brief description of the Button component.

**Version v1.0.0**

-  All variants in Primary, Secondary, Tertiary, Link (expect inline links)
-  All sizes like sm, md, lg, xl, 2xl.

#### **Available subVariants in each Hierarchy**

| hierarchy   | subVariant                                              |
| ----------- | ------------------------------------------------------- |
| `PRIMARY`   | PRIMARY, DESTRUCTIVE                                    |
| `SECONDARY` | PRIMARY, DESTRUCTIVE, DESTRUCTIVE_OUTLINE, GRAY_OUTLINE |
| `TERTIARY`  | PRIMARY, DESTRUCTIVE, GRAY                              |
| `LINK`      | PRIMARY, DESTRUCTIVE, GRAY                              |
